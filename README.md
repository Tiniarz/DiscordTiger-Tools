🐯 Introduction

DiscordTiger is a high-performance terminal user interface (TUI) designed for advanced server management, audit-log testing, and stress-testing server infrastructure. By utilizing a hybrid engine of for bot-level integration and raw for user-level API interaction, DiscordTiger provides a comprehensive suite of 74 operational tools.discord.pyrequests

🛠 Features
Dual-Token Architecture: Seamlessly handles both Bot and User tokens with automatic header switching.

Silent Configuration: Automatically assigns credentials from a local file to streamline operations.Token.txt

Dynamic TUI: Features an immersive Green Matrix rain sequence followed by a red-themed ASCII command center.

Tiered Operations:

Nuke [01-25]: Total environmental reset and purge modules.

Raid [26-50]: Communication and spam stress-testing infrastructure.

ULTRA [51-74]: The most aggressive suite of tools utilizing high-frequency API requests for maximum server impact.

Quick Start
Clone the environment: Ensure all files are placed in a dedicated folder within your VM.

Assign Credentials: Create a file named in the root directory and paste your token (Bot or User) inside.Token.txt

Execute: Run the script via your terminal:

Bash
python DiscordTiger.py
